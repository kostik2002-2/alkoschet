# Алкосчёт

Алкосчёт — статический сайт-калькулятор для расчёта алкоголя и безалкогольных напитков на свадьбу.

Сайт опубликован здесь:

https://alkoschet.cinemaprompt.ru/

## Что умеет

- рассчитывает шампанское, вино, крепкий алкоголь, воду, сок и пиво;
- учитывает количество гостей, непьющих гостей и детей;
- учитывает количество пьющих мужчин;
- поддерживает разные форматы банкета: спокойный, средний и хардкорный;
- адаптирован для мобильных устройств;
- работает без сервера и базы данных.

## Структура проекта


## Публикация

Проект хранится в GitHub:

https://github.com/kostik2002-2/alkoschet

Cloudflare автоматически публикует сайт после команды:

```bash
git push
```

Основная ветка:

```text
main
```

## Рабочий процесс

Обычный порядок работы:

```bash
git status
git add .
git commit -m "Краткое описание изменений"
git push
```

После `git push` Cloudflare запускает новый деплой и обновляет сайт.
